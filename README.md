# Lite Auth
LiteAuth是一个基于LDAP精简的身份认证系统。
它针对用户管理，身份认证场景进行了优化，让你能更方便的接入LDAP，管理用户。

# 特性

* 精简了大部分LDAP的内容，无需学习复杂的LDAP体系，直接使用。
* 自带管理后台，添加修改用户更方便。
* 自带安全策略（多次登录失败锁定策略，密码过期策略，密码复杂度校验策略）
* 成员可自主登录管理后台修改密码
